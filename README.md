## 🎯 How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/ayushborban09-lab/Fullstack-webdev.git
   ```

2. Open any HTML file in your browser:
   - Navigate to a project folder
   - Double-click the `.html` file or use `Live Server` extension in VS Code

3. Explore the code:
   - Each project is self-contained in its own folder
   - Open `.html`, `.css`, and `.js` files to see the source code
